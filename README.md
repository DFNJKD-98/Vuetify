# learn_vuetify

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

- 在线访问地址： http://sq.dfnjkd.fun

- 目前项目提供书签访问功能（bookMark）
